SQL Python Analysis

Combining Python and SQL to analyze hotel booking data. 
The CSV dataset is loaded directly into a SQLite database using pandas, then queried using SQL through Python.

Key findings:

Portugal is the top booking country with 48,590 bookings
Group bookings have the longest average lead time at 187 days
SQL queries and pandas groupby produce identical results

Tools used: Python, pandas, sqlite3, SQLAlchemy, Jupyter Notebook
